# NewsApp
### Android App which displays latest news from **newsapi.org**

Android project with the usage of libraries</br>
 **RxJava**</br>
 **Dagger2**</br>
 **Retrofit**</br>
 **Picasso**</br>
 **ViewModel**</br>
